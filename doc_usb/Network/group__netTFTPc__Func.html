Description

TFTP Client is used to connect to a TFTP Server for simple file transfer.

A TFTP Client can exchange files with a TFTP Server. File delete, file move, and file rename are not impossible, because the TFTP protocol does not support these file operations in general.

This documentation is separated as follows:

  • Control Interface explains how to start the TFTP Client.
  • User Callback describes the operation of a TFTP client session and how an event notification is sent to the user application.
  • File System Interface gives you details about the functions that are used to read/write data on the TFTP server's storage device.
  • Configuration explains the configuration options of the TFTP client.
